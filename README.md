# Mental Health Aider Platform

A data-driven platform that helps organizations monitor employee wellbeing and detect burnout risks using analytics.

## Architecture

Event Generator → Spark Processing → Analytics Database → Spring Boot API → Angular Dashboard

## Tech Stack

Spark (PySpark)
Spring Boot
Angular
PostgreSQL
Docker
Kubernetes

## Features

Employee mood tracking
Burnout risk scoring
Department stress analytics
High-risk employee alerts
Real-time dashboards
